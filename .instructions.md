# Custom Instructions for GitHub Copilot in SocialWork Project

## Project Overview
This workspace contains a tech news aggregation system with agents that pull news from RSS feeds and generate digests (daily, weekly, monthly) and social media posts (Instagram carousels). The system uses static site generation with Eleventy.

## Key Directories
- `agents/`: Markdown files describing agent behaviors (daily digest, weekly, monthly, IG post)
- `digests/`: Output files for digests (flat for daily, subdirs for weekly/monthly)
- `posts/`: Instagram carousel outputs
- `digests.11tydata.js`: Data file for Eleventy
- `_site/`: Generated static site

## Agent Pipeline
When running the full pipeline:
1. Daily Digest: Fetches RSS feeds for last 48 hours, generates digests/YYYY-MM-DD_HHMM_tech-news.md
2. Weekly Digest: Aggregates last 7 days' daily digests into digests/weekly/YYYY-MM-DD_HHMM_tech-news-weekly.md
3. Monthly Digest: Aggregates last 30 days' digests (prefers weekly) into digests/monthly/YYYY-MM-DD_HHMM_tech-news-monthly.md
4. IG Post: Summarizes last 24 hours into posts/YYYY-MM-DD_HHMM_tech-news-instagram.md

## File Naming Conventions
- All output files start with YYYY-MM-DD_HHMM (24-hour time)
- Never overwrite; append -2, -3 if needed
- Retention: Daily 30 days, Weekly 60 days, Monthly 365 days, Posts 30 days

## Source List
20 RSS feeds defined in agents/tech-news-digest.md, covering general, developer, AI, security, hardware sources.

## Behavior Notes
- Daily agent fetches RSS; others read-only from local files
- Dedupe stories across sources
- Group by categories: AI, Security, Hardware, etc.
- Flag rumors/unverified
- Print output paths after writing

## Commands
- Build site: `npm run build` (Eleventy)
- Serve site: `npm run serve` (Eleventy with watch)
